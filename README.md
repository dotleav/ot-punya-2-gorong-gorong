# Medical Quiz App

A single-page medical quiz application for Indonesian medical education, covering topics in Geriatrics, Endocrine-Metabolic, Nutrition, Clinical Pharmacology, Neurology, and Occupational Medicine.

## Key Technologies

- **TanStack Start** with React 19 — SSR-capable full-stack framework
- **TailwindCSS v4** — utility-first styling
- **TypeScript** — type-safe components
- **Netlify** — hosting and deployment

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Features

- 111 multiple-choice questions (5 options each) with randomized answer positions
- Category filter (Endokrin-Metabolik, Geriatri, Gizi & Nutrisi, Farmakologi Klinik, Neurologi-Geriatri, Kedokteran Kerja, and image-based variants)
- Per-question explanations shown after answering
- Timer and score tracking
- Results screen with wrong-answer review
- Full review mode with navigation through all questions
